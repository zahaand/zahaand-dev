## About

Static portfolio website built as a single self-contained `index.html` file — no frameworks, no build step, no external dependencies. All assets (photo, screenshots) are Base64-embedded.

**Live:** [zahaand.dev](https://zahaand-dev.vercel.app)

## Features

- 🌙 / ☀️ Dark / light theme toggle with system preference detection
- 🇷🇺 / 🇬🇧 RU / EN language switch — full content translation
- 💻 Interactive terminal with 9 commands
- 🐛 Hidden game — catch bugs before they reach production (`play` in terminal)
- 🎮 Konami code easter egg (↑↑↓↓←→←→BA)
- 🔍 SEO-optimized: meta tags, Open Graph, Twitter Card, Schema.org JSON-LD
- 📱 Fully responsive — mobile (375px), tablet, desktop

## Terminal Commands

| Command | Description |
|---------|-------------|
| `help` | List all available commands |
| `whoami` | About me |
| `skills` | Full tech stack |
| `projects` | Portfolio projects with GitHub links |
| `services` | What I can build for you |
| `contact` | Phone, Telegram, Email, GitHub |
| `fact` | Random fact from my projects |
| `play` | 🐛 Hidden game: catch bugs before prod |
| `clear` | Clear the terminal |

**DevTools easter egg:** open browser console and run `hire()`.

# zahaand.dev — Сайт-портфолио

Персональный сайт-портфолио Захаренкова Андрея, Java Backend разработчика, открытого для фриланс-проектов.

## О сайте

Статический сайт-портфолио в виде единого самодостаточного файла `index.html` — без фреймворков, без сборки, без внешних зависимостей. Все ресурсы (фото, скриншоты) встроены как Base64.

**Сайт:** [zahaand.dev](https://zahaand-dev.vercel.app)

## Возможности

- 🌙 / ☀️ Переключение тёмной/светлой темы с определением системных настроек
- 🇷🇺 / 🇬🇧 Переключение языков RU / EN — полный перевод контента
- 💻 Интерактивный терминал с 9 командами
- 🐛 Скрытая игра — лови баги до того как они уйдут в прод (команда `play` в терминале)
- 🎮 Пасхалка Konami code (↑↑↓↓←→←→BA)
- 🔍 SEO-оптимизация: meta-теги, Open Graph, Twitter Card, Schema.org JSON-LD
- 📱 Полная адаптация — мобильные (375px), планшет, десктоп
