Последнее обновление: апрель 2026  ·  zahaand.dev

Общие положения

Настоящая политика описывает, какие данные собираются при посещении сайта zahaand.dev, как они используются и защищаются. Сайт является личным портфолио Захаренкова Андрея Игоревича.

Сбор данных

Сайт не содержит форм регистрации и не собирает персональные данные напрямую. При посещении автоматически могут собираться технические данные:

  • IP-адрес и данные браузера (в целях аналитики)
  • Страницы и разделы, которые вы посещаете
  • Время и продолжительность визита
  • Источник перехода на сайт

Файлы cookie

Сайт использует файлы cookie в следующих целях:

  • localStorage — сохранение выбранной темы (тёмная / светлая)
  • localStorage — сохранение выбранного языка (RU / EN)
  • localStorage — сохранение факта принятия политики cookie
  • Яндекс.Метрика — анонимная веб-аналитика: посетители, источники, поведение

Данные Яндекс.Метрики обрабатываются согласно политике конфиденциальности Яндекса. Отключить сбор можно через opt-out плагин.

Передача данных третьим лицам

Данные не продаются и не передаются третьим лицам, за исключением Яндекс.Метрики, обрабатывающей их согласно своей политике.

Внешние ссылки

На сайте есть ссылки на внешние сервисы (Telegram, Gmail, GitHub). При переходе вы покидаете сайт — эти сервисы имеют собственные политики конфиденциальности.

Права пользователей

В соответствии с 152-ФЗ «О персональных данных» вы вправе:

  • Запросить информацию о хранящихся данных
  • Потребовать удаления данных
  • Отозвать согласие на обработку

Изменения политики

Актуальная версия всегда доступна на zahaand.dev/privacy.html. Продолжение использования сайта означает согласие с обновлениями.

Privacy Policy

Last updated: April 2026  ·  zahaand.dev

General

This policy describes what data is collected when visiting zahaand.dev, how it is used and protected. The website is a personal portfolio of Andrei Zakharenkov.

Data Collection

The site does not contain registration forms and does not collect personal data directly. The following technical data may be collected automatically:

  • IP address and browser data (for analytics purposes)
  • Pages and sections you visit
  • Visit time and duration
  • Traffic source

Cookies

The site uses cookies for the following purposes:

  • localStorage — saving the selected theme (dark / light)
  • localStorage — saving the selected language (RU / EN)
  • localStorage — saving cookie consent
  • Yandex.Metrika — anonymous web analytics: visitors, sources, behavior

Yandex.Metrika data is processed according to Yandex's privacy policy. You can opt out via the opt-out browser plugin.

Third Party Data Sharing

Data is not sold or transferred to third parties, except for Yandex.Metrika which processes it under its own policy.

External Links

The site contains links to external services (Telegram, Gmail, GitHub). When you follow them you leave this site — those services have their own privacy policies.

User Rights

You have the right to:

  • Request information about stored data
  • Request deletion of data
  • Withdraw consent to data processing

Policy Updates

The current version is always available at zahaand.dev/privacy.html. Continued use of the site means agreement with any updates.
